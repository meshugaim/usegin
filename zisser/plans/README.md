# plans/

Plans Zisser is building. One file per plan.

For shipping work that needs Linear, the plan still lives here as a working
draft until it's distilled into Linear issues via `plan create`. Cross-link
between this file and the Linear issue once created.

For usegin-grade or zisser-internal work, the plan lives here permanently
(z024 — not Linear-everything).
