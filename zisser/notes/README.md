# notes/

Notes Zisser writes *for* Lihu. Briefings, summaries, recaps, follow-up
reminders, "things to know" digests.

This is Zisser → Lihu, not Lihu → Zisser. (Lihu → Zisser raw goes in
`inbox/`; placed thoughts go in `usegin/zettel/`, Linear, etc.)

## Subfolders

- `people/<name>.md` — per-person follow-ups and threads. Open one when Lihu
  mentions someone he wants to track. Keep it tight — recent threads at top.
- `recaps/<YYYY-MM-DD>.md` — when Lihu asks "what shipped this week" or
  similar. One file per recap.
