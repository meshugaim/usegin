---
title: See Linear issues from a specific session
handle: plan-session-filter
tags: [linear, sessions, tracking]
context: When tracing what work happened in a Claude session
---

`plan list --session <id>` filters Linear issues to only those created
or updated during a specific Claude session. Pairs well with `session list`
to find the session ID first.

Tip: `session list` shows IDs — copy the 8-char prefix to use here.
