---
title: Browse and re-run past bash commands
handle: session-bash
tags: [sessions, bash, fzf]
context: When looking for a command you ran previously
---

`session bash` opens an fzf browser of bash commands from past sessions.
Press Ctrl-R to run a command directly from the browser, or Enter to
copy it.

Add a session ID to scope: `session bash 502de9c7`, or `--grep` to filter.
