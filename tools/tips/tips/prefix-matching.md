---
title: All CLIs support prefix matching
handle: prefix-matching
tags: [cli, productivity, shortcuts]
context: When using any CLI tool in this repo
---

Type `plan l` instead of `plan list` — all CLIs in this repo use
Commander.js which supports unambiguous prefix matching. Saves
keystrokes on every command.

Works with: `tip s` (search), `docs l` (list), `session f` (find), etc.
