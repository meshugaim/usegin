## How to add a new "thing we grow"

1. Notice friction caused by a missing artifact (z009 — friction loop).
2. Add a row to **Open-to-empty** above with a guessed location and what it would accumulate.
3. Create the empty file (literally `touch` or write a one-line README).
4. If the artifact starts getting entries, promote the row to **Active**.
5. If after a few sessions it never gets used, leave it — it costs nothing to keep an empty address (z003).
