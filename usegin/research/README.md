# Research

Cross-cutting investigations that don't belong to any one sub-app. Each topic
is its own folder; when a topic has matured to "we have a recommendation",
it either:

- Lifts into a zettel (atomic) under `usegin/zettel/zettels/`, or
- Lifts into a sub-app of its own under `usegin/<topic>/`, or
- Closes out with a decision note here and an outbound link.

Append-mostly. Don't delete folders when they conclude — leave the trail.

## How to add a topic

1. Notice a question that doesn't fit a sub-app and is too big for a zettel.
2. Create `usegin/research/<topic>/` with a `README.md` stating the question
   and what answering it would unblock.
3. Use the folder freely — whiteboards, sub-investigations, manufactured
   examples, dialogue logs.
4. When it concludes, write the conclusion note. Don't delete the folder.
