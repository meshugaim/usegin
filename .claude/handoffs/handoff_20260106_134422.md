# Figma Implement

Implement UI changes to make the app match Figma designs. Scope: `/workspaces` routes only.

## Before You Start

### 1. Feature Flag Required

All new UI is behind a feature flag. You MUST enable it:

**Client-side (cookie):**
```
effi-new-ui=true
```

**Server-side check:**
```typescript
import { isNewUIEnabledServer } from '@/lib/feature-flags-server'
```

**Enable via UI:**
1. Navigate to `http://localhost:3000/toggles`
2. Enable "New UI Design" toggle
3. Then navigate to `/workspaces` routes

### 2. Scope

Only edit these routes:
- `/workspaces` - Workspaces list
- `/workspaces/[workspaceId]` - Workspace detail
- `/workspaces/[workspaceId]/settings` - Workspace settings
- `/toggles` - Feature toggles page

Leave all other routes untouched.

### 3. Orient on the App

**File tree (scope highlighted):**
```

│   ├── workspaces/               # ⭐ EDIT THESE
```

   - With `effi-new-ui=true` (new design)
   - With `effi-new-ui=false` (old design still works)

### 6. Track

1. Update diffs.json entry status
2. Commit with Linear issue reference
3. Close Linear issue if complete
