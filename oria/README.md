# oria/

Oria's personal area inside the usegin monorepo. Two life-areas:

- **`life/`** — personal-machine projects and setups (this PC, home lab, devices).
- **`work/`** — Effi / askeffi.ai work.

Each project gets its own subdir with a `README.md`, its canonical scripts, and
pointers to anything that lives outside the repo (system services, GNOME
settings, patched apps). The repo is the source of truth — live copies under
`~/.local/bin` etc. are symlinks back here where practical.
